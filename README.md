# detach-mta-dashboard

To compare space saved.

## Installation

```
git clone https://github.com/Naedri/detach-mta-board.git
cd detach-mta-board/
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
